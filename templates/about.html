About This To-Do App

A simple and clean To-Do application built using Flask, SQLAlchemy, and Bootstrap.

What This Website Does

This web application allows you to create and manage a list of tasks quickly and easily. It is designed for beginners learning Flask and web development.

Features You Can Use

  • Add new tasks using the form on the homepage.
  • View all your tasks displayed in a neat table.
  • Update tasks
  • Delete tasks permanently from your list.
  • Stores data using a database SQLite .

To Use The Website

  1. Go to the Home page.
  2. Enter your task title and description.
  3. Press Add Task to save it.
  4. Your task will appear in the list below.
  5. Use Update or Delete.

Technology Used

  • Python Flask
  • Flask SQLAlchemy
  • Bootstrap 5
  • HTML + CSS

Made by Mahi while learning Flask.
